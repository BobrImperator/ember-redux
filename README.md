# Ember Redux

[![Travis][build-badge]][build] [![npm package][npm-badge]][npm]

## Description

[ember-cli][] addon that provides simple [redux][] bindings for [ember.js][]

## Installation

```
npm install ember-browserify -D
npm install ember-redux -D
```

## Documentation

http://www.ember-redux.com/

## Running Tests

    npm install
    bower install
    ember test
